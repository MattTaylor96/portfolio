Invoice Validation

This application uses local storage to save entered Client details, then validates a number of invoice files against these details and a number of other conditions. The application uses the PapaParse plugin to convert .csv files to JSON, then validates the lines within the files against a set number of conditions. The user can export any records that did not meet the conditions to a .txt file as an error log. This was built to replace a time-consuming task of manually checking invoice lines, and is used regularly in a professional capacity.
